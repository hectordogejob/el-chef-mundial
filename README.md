# 🍳 El Chef Mundial

App de cocina internacional con inteligencia artificial. Aprende a cocinar platillos de los 5 continentes con el **Chef Vittorio**, tu mentor personal.

## ✨ Características

- 🤖 **Chef con IA** - Chat con un chef profesional que responde cualquier pregunta de cocina
- 🌍 **42 platillos** cocinas de 6 continentes
- 🎮 **Gamificación** - Sube de nivel de Aprendiz a Master Chef con XP y logros
- 📖 **Catálogo** con filtros por continente, país y presupuesto
- ❤️ **Favoritos** y lista de ingredientes con compartir por WhatsApp
- 💬 **Múltiples conversaciones** con memoria (como ChatGPT)
- 👑 **Modelo freemium** - 3 preguntas gratis por día, premium ilimitado
- 📱 **PWA** - Instálala en tu celular como app nativa

## 🛠️ Stack Tecnológico

- **Backend:** Python 3.14 + FastAPI + SQLAlchemy
- **Frontend:** React + Vite + Axios
- **Base de datos:** SQL Server
- **IA:** Claude API (Anthropic)
- **Autenticación:** JWT + bcrypt
